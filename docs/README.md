# HamsterNET

> Eine künstliche Intelligenz Library für den Hamstersimulator 

# Was macht diese Library?
Diese Library implementiert simple selbstlernende Algorithmen und ein objektorientiertes Interface für diese.

Sie wurde explizit für die Nutzung mit dem [Java Hamster Modell von Dietrich Boles](http://www.java-hamster-modell.de/index2.html) geschrieben, um das Konzept der Simplizität, welches das Java Hamster Modell verfolgt, fortzuführen und Programmieranfängern einen einfachen Einstieg in die Welt der künstlichen Intelligenz zu bieten.


# Was bietet diese Library?

- eine Implementierung eines [neuronalen Netzes](https://de.wikipedia.org/wiki/Neuronales_Netz)
- eine Implementierung des Crossentropy Algorithmus
